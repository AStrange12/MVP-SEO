# **App Name**: LocalBoost AI

## Core Features:

- AI Business Generator: Generate comprehensive business details including description, tagline, SEO keywords, and a services list based on user inputs (Business Name, Category, Location) using a mock AI tool.
- Landing Page Preview Generator: Automatically generate and display a visually clean and modern landing page preview featuring essential sections like Hero, About, Services, and Contact, populated with the AI-generated business information.
- AI Marketing Content Generator: Generate engaging marketing content, such as Instagram captions and WhatsApp promotional messages, to help businesses attract customers using an AI tool.
- Website Deployment Simulation: A 'Deploy Website' button that simulates a successful website deployment with a mock success message, providing a sense of completion for the user.
- Loading Animations: Smooth and aesthetically pleasing loading animations implemented during AI generation and page transitions to enhance user experience.
- Dark Mode Toggle: Implement a user-friendly dark mode toggle for comfortable viewing and to align with modern SaaS aesthetic.

## Style Guidelines:

- Primary accent color (for interactive elements and highlights): A vibrant, tech-inspired purple, #6B2EDB (HSL: 270, 70%, 60%). This color signifies innovation and sophistication against the dark theme.
- Background color: A deep, subtly purple-tinted charcoal gray, #19151C (HSL: 270, 15%, 10%). This provides a strong foundation for the dark mode, evoking professionalism and a sleek aesthetic.
- Accent color (for secondary highlights and energetic calls to action): A bright, cool-toned blue, #ADBCFC (HSL: 240, 90%, 70%). This analogous color provides a striking contrast and enhances visual hierarchy.
- Headlines and prominent text: 'Space Grotesk' (sans-serif) for a modern, techy, and distinctive look, emphasizing the 'AI' aspect of the app.
- Body text and functional descriptions: 'Inter' (sans-serif) for its high readability, objective feel, and excellent legibility across different screen sizes, ensuring clarity for generated content.
- Utilize clean, modern, and minimalistic line icons that complement the SaaS aesthetic. Icons should be clear and functionally oriented, maintaining a sophisticated appearance.
- Design with clean cards and sections, featuring subtle gradient backgrounds and ample whitespace to create a structured and premium SaaS-style interface. Sections should flow logically and intuitively.
- Implement smooth, subtle transitions and animations throughout the UI, including element hover states, content loading, and navigation. Use animated progress indicators to provide user feedback during AI generation.
